$categories = Cache::rememberForever('categories:tree', function () {
    return Category::with('children')
        ->whereNull('parent_id')
        ->where('is_active', true)
        ->orderBy('sort_order')
        ->get();
});

✅ Kina use gareko?

Category tree:

Frequently used (header menu, sidebar, filters)

Rarely changed

Hierarchical structure (parent → children)

So this is perfect candidate for caching.

Without cache:

10,000 DB queries/hour

➡️ Massive performance improvement
➡️ DB load drastically reduce
